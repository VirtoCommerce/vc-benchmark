# har2ammo 
=============
https://github.com/banzalik/har2ammo

Transform [.har](http://en.wikipedia.org/wiki/.har) files to [ammo.txt](https://yandextank.readthedocs.org/en/latest/tutorial.html#uri-style-uris-in-file) for [yandex-tank](https://github.com/yandex-load/yandex-tank).

## Install

`har2ammo` requires [nodejs](http://nodejs.org/) èand[npm](https://npmjs.org).

Install:

`npm install -g har2ammo`

## Using

`har2ammo -c config.js -i test.har -o ammo.txt`
